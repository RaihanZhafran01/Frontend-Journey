## 🎨 Apa itu CSS?
**CSS (Cascading Style Sheets)** adalah bahasa desain yang digunakan untuk mengontrol presentasi visual dari dokumen yang ditulis dalam bahasa markup seperti HTML. Jika HTML adalah kerangka atau struktur dari sebuah bangunan, maka CSS adalah arsitek dan dekorator interiornya.

CSS memungkinkan pemisahan antara konten (HTML) dan desain (Visual). Hal ini memberikan fleksibilitas untuk mengubah tampilan situs secara menyeluruh tanpa harus memodifikasi strukturnya, serta memastikan situs web dapat diakses secara optimal di berbagai perangkat dan ukuran layar.

## 🧠 Materi Inti & Fondasi Utama
Berikut adalah pilar-pilar penting dalam CSS yang didokumentasikan dalam repositori ini:

### 1. Selektor dan Spesifisitas
Memahami bagaimana menargetkan elemen HTML secara presisi menggunakan *class*, *ID*, *attribute selectors*, serta memahami aturan tingkatan prioritas (*specificity*) agar kode CSS tetap bersih dan terukur.

### 2. Box Model
Konsep fundamental yang menganggap setiap elemen sebagai kotak persegi. Mempelajari interaksi antara **Content**, **Padding**, **Border**, dan **Margin** yang krusial untuk mengatur jarak dan ukuran elemen secara akurat.

### 3. Tata Letak Modern (Layout)
* **Flexbox:** Teknik *one-dimensional* untuk menyusun elemen secara fleksibel dalam baris atau kolom.
* **CSS Grid:** Sistem *two-dimensional* yang memungkinkan pembuatan desain tata letak yang kompleks dan presisi tinggi.

### 4. Desain Responsif & Media Queries
Implementasi strategi *Mobile First Design* untuk memastikan antarmuka pengguna beradaptasi secara sempurna terhadap berbagai resolusi layar, dari ponsel pintar hingga layar monitor *ultrawide*.

### 5. Animasi dan Transisi
Pemanfaatan `@keyframes` dan properti `transition` untuk menciptakan pengalaman pengguna yang interaktif, halus, dan dinamis tanpa harus bergantung sepenuhnya pada JavaScript.
